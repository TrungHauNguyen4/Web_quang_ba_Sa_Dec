Design a modern, responsive tourism website UI for a local destination in Vietnam: Sa Đéc city, Đồng Tháp province.

Style:

* Clean, modern, friendly, and vibrant
* Inspired by tourism and cultural websites
* Use soft colors: green, blue, orange (representing nature, river, and local culture)
* Use card-based layout with rounded corners and subtle shadows
* Mobile-first responsive design

Tech context:

* This design will be implemented using React and Next.js
* Use reusable components and clear layout structure
* Follow best practices for web UI/UX and SEO-friendly structure

---

Main Pages & Sections:

1. Home Page:

* Hero banner with large background image of Sa Đéc (flowers village, river view)
* Title: "Khám phá Sa Đéc - Thành phố hoa của Đồng Tháp"
* Call-to-action buttons: "Khám phá ngay", "Lên lịch trình"
* Featured sections:

  * Popular destinations (card grid)
  * Local cuisine highlights
  * Latest news/events
* Quick navigation icons (Địa danh, Ẩm thực, Bản đồ, Tin tức)

---

2. Destinations Page (Địa danh):

* Grid layout of destination cards
* Each card includes image, title, short description
* Categories filter: Văn hóa, Du lịch, Tâm linh
* Detail page:

  * Large image gallery
  * Description
  * Location map
  * Visiting information

---

3. Cuisine Page (Ẩm thực):

* Food cards with image, name, description
* Highlight local specialties:

  * Hủ tiếu Sa Đéc
  * Nem Lai Vung
  * Bánh phồng tôm
* Optional: location or restaurant suggestions

---

4. Map Page:

* Interactive map layout (Google Maps style placeholder)
* Markers for:

  * Tourist attractions
  * Food places
  * Public offices

---

5. News Page (Tin tức):

* List of articles with thumbnails
* Categories: Tin mới, Sự kiện, Thông báo
* Article detail page with content and images

---

6. About Page (Giới thiệu):

* Overview of Sa Đéc:

  * History
  * Culture
  * Geography
* Include images and icons

---

7. Gallery Page (Thư viện):

* Image gallery grid
* Video section (YouTube style thumbnails)

---

8. Contact Page (Liên hệ):

* Contact form (name, email, message)
* Local authority information (UBND)
* Embedded map

---

9. Administrative Services Page (Dịch vụ hành chính):

* List of services (cards)
* Form UI for submitting applications
* Status tracking UI

---

Components to include:

* Navbar (logo + menu)
* Footer (contact info + social icons)
* Cards (reusable)
* Buttons (primary, secondary)
* Form inputs
* Tabs / filters

---

Design Requirements:

* Use consistent spacing and typography
* Clear visual hierarchy
* Use icons for categories
* Optimize for accessibility and readability
* Provide both desktop and mobile layouts

---

Output:

* High-fidelity UI design
* Organized components (design system ready)
* Clearly structured pages for easy conversion to Next.js
